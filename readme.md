Blazingly fast world manipulation with FastAsyncWorldEdit in Skript.\
This addon adds Skript syntax to use regions, patterns, schematics and more!

</div>

### Requirements
- FastAsyncWorldEdit
- Skript 2.14+
- Paper 1.21+
- Java 21+

### Features
- Patterns
- Regions (cuboid, cylinder, ellipsoid, and polygon)
- Masks
- Shapes (lines, splines, blobs, circles, cones, cylinders, pyramids, and spheres)
- Schematics (save & paste)

Head over to the [wiki](https://modrinth.com/plugin/skriptworldedit) for a detailed walkthrough of the syntax and a few examples!
